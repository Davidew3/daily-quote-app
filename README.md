# Daily Quote App

## Description
This is a simple React-Bootstrap application that displays a daily quote to the user. It is one of the five capstone projects for the Frontend Development Libraries Certification.

## Installation
To install the application, follow these steps:

1. Clone the repository: `git clone https://github.com/Davidew3/daily-quote-app.git`
2. Navigate into the project directory: `cd daily-quote-app`
3. Install the dependencies: `npm install`

## Usage
To start the application, simply run `npm start` from the project directory.
